TMDb API Key

Enter your TMDb API key (v3 auth). Get one at themoviedb.org/settings/api .
Can also be set via the TMDbApiKey environment variable.

Tag Name

The name of the tag this plugin adds or removes based on physical release status. Defaults to Hidden.

Auto-Scan

When enabled, newly added movies and series are processed immediately using per-library settings. Disabled libraries and excluded items are still skipped.

Scan Schedule

The hour of day when the scheduled scan runs. Changes are applied when you click Save.

Dry-Run Mode

When enabled, scans will log what they would do without actually adding or removing tags. Useful for previewing changes before committing them. Auto-scan for new items is not affected.

Preferred Region

When set, only physical releases in the selected country are considered. Leave on "All countries" to check worldwide release data. The list is populated from TMDb. Changes are applied when you click Save.

Libraries

Configure per-library settings. Disabled libraries are skipped entirely. Set a custom tag name per library, or leave blank to use the global tag above.

Excluded Movies and Series

Selected movies and series are skipped entirely by the scheduled scan. The plugin will not add or remove the configured tag for those items.

Parental Control

When enabled, newly created users will automatically have the configured tag (e.g. Hidden) added to their Parental Control BlockedTags. This ensures new users can never see tagged content without manual admin intervention.

Adds the configured tag to the BlockedTags of every user who doesn't already have it. Use this to retroactively apply parental control to existing accounts.
